# Changelog

All notable changes to the "gemini-beads" extension will be documented in this file.

## [0.3.0] - 2026-02-25

### Added
- New `decision` command for recording and tracking project decisions.
- Updated command definitions for improved AI tool-calling accuracy.

### Changed
- Refactored `bd prime` hook output to be more concise, reducing token usage.
- Aligned documentation with Dolt-based architecture and updated the "claim" workflow.

## [0.2.0] - 2026-01-31

### Added
- Integrated Beads as a Skill for Gemini.
- Added session hooks for automatic `bd prime` execution.
- Expanded documentation with detailed resources for advanced features.

## [0.1.0] - 2026-01-18

- Initial release of the Gemini Beads extension.
- Wraps `bd` CLI commands for issue tracking.
- Includes `GEMINI.md` for AI context awareness.
